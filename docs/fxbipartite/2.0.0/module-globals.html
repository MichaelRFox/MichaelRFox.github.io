Properties:
Name Type Description
graph biPartite

The biPartite class object. This property is set when the biPartite constructor is called. Default: undefined.
data Array

A two-dimensional array of source (string), target (string), and value (number) tuples. Default: [].
container HTML.div

The element in which the graph is to be rendered. Default: undefined.
containerBackground string

The background color of the container. Default: 'rgb(255, 255, 255)'.
labelColor string

The color of the labels. Default: 'rgb(0, 0, 0)'.
fillColors Array

An array of strings for the colors of the bars and edges. Default: ['rgb(102, 194, 165)', 'rgb(252, 141, 98)', 'rgb(141, 160, 203)', 'rgb(231, 138, 195)', 'rgb(166, 216, 84)', 'rgb(255, 217, 47)', 'rgb(229, 196, 148)', 'rgb(179, 179, 179)'].
lightTextColor string

The text color to use over dark backgrounds. Default: '(rgb(255, 255, 255)'.
lightTextLuminance number

The luminance of the lightTextColor. Default: 1.
darkTextColor string

The text color to use over light backgrounds. Default: '(rgb(0, 0, 0)'.
darkTextLuminance number

The luminance of the darkTextColor. Default: 0.
sort string

The sorting method for the bars. One of [alpha | barycentric | sh | none]. Default: 'alpha'.
sourceKeys Array

The text labels for the source bars. Default: [].
targetKeys Array

The text labels for the target bars. Default: [].
orient string

The orientation of the graph. One of [vertical | horizontal]. Default: 'vertical'.
pad number

The amount of padding in pixels between each bar. Default: 1.
duration number

The duration of transitions in milliseconds, Default: 750.
edgeOpacity number

The opacity of the edges [0..1]. Default: 0.4.
edgeMode string

How the edges are to be rendered. One of [curved | straight]. Default: 'curved'.
refresh boolean

True of the graph needs to be drawn, false otherwise. This property is set internally and is not exposed by the biPartie class object. Default: true.
width number

The width of the container in pixels. This property is set internally by the containerListener function and is not exposed by the biPartie class object. Default: 0.
height number

The height of the container in pixels. This property is set internally by the containerListener function and is not exposed by the biPartie class object. Default: 0.
minWidth number

The minimum width of the bars to accommodate label size and font style. This property is set internally by the getMargins function and is not exposed by the biPartie class object. Default: 0.
minHeight number

The minimum height of the bars to accommodate label size and font style. This property is set internally by the getMargins function and is not exposed by the biPartie class object. Default: 0.
labelOffset number

How far the label is offset from the edge of each bar. Default: 0;
eventTypeOver string

The type of event which triggers interaction. One of ['mouseenter' | 'click' | 'dblclick']. This property is set through the biPartie class object's event method. Default: 'mouseenter'.
eventTypeOut string

The type of event which un-triggers interaction. One of ['mouseleave' | 'click' | 'dblclick']. This property is set through the biPartie class object's event method. Default: 'hover'.
eventListenerOver function

The function called by the event listener which triggers interaction.One of [mouseOver | click]. This property is set through the biPartie class object's event method. Default: mouseover.
eventListenerOut function

The function called by the event listener which un-triggers interaction. One of [mouseOut | click]. This property is set through the biPartie class object's event method. Default: mouseOut.
