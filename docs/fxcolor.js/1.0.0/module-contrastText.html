Computes the text color (white or black) with the highest contrast with the fxColor class object's color. See WCAG 2.0.

Parameters:
Name Type Description
format String

The desired format to return the fxColor class object's color. One of ['cmyk' | 'hex' | 'hsl' | 'hsv' | 'hwb' | 'lab' | 'lch' | rgb' | 'int']. Default: 'rgb'.
Returns:

A W3C-formatted color specification of the requested forat for either white or black.
